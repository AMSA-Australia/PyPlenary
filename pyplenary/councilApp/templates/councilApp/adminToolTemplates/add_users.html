Admin Centre - Add Users

Invite new users to register accounts on the Webapp. Users are emailed an invitation link to register with the Webapp.

You MUST use this CSV template to upload users. Click on the link to download the template.

Some important notes on using the template:

  • Institution names must be exactly one from this list (eg. University of Melbourne and UMelb are valid, but Melbourne University is invalid.)
  • In the column for first-time attendee status, type the number 1 for first-timers, and type 0 otherwise.
  • Do not add or remove columns from the template.
  • By default, users with pending invitations or registered accounts are not reissued an invitation link. However, the checkbox below allows you to force the reissuing of invitations (eg. if a user claims not to have received their original invitation)
